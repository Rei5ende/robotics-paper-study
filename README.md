## Research Themes

Reading these papers, a common theme emerges across Robot AI research:

> Rather than oversimplifying the input, preserve its richness throughout the entire process
> and make decisions incrementally.

```
LINGO-Space  → spatial instructions  → probabilistic incremental grounding
DiSPo        → coarse demonstrations → fine-grained action control
Context-Nav  → text descriptions     → context-driven exploration
```
